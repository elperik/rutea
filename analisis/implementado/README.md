# Implementado

Esta carpeta conserva la memoria técnica de iniciativas terminadas.

Cada documento debe permitir que otro agente determine:

- qué existe realmente;
- dónde está implementado;
- cómo se verifica;
- qué contratos o datos utiliza;
- qué limitaciones permanecen;
- qué trabajo posterior depende de él.

No modificar retrospectivamente una decisión histórica sin dejar constancia. Cuando una implementación sea sustituida, actualizar el documento original con una referencia al nuevo documento que la reemplaza.
