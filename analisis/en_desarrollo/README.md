# En desarrollo

Esta carpeta contiene exclusivamente iniciativas activas.

Antes de añadir un documento aquí:

- debe existir previamente en `pendiente/`;
- deben estar definidos alcance y criterios de aceptación;
- debe existir un responsable;
- las dependencias bloqueantes deben estar resueltas o declaradas;
- el plan de pruebas debe ser suficiente para el riesgo.

Durante el trabajo, actualizar el documento con decisiones duraderas, checklist y evidencia. No introducir aquí análisis generales no vinculados a una implementación activa.
