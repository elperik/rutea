# Pendiente

Esta carpeta contiene trabajo analizado que todavía no está en ejecución.

Un agente puede ampliar estos documentos, resolver alternativas o preparar pruebas sin moverlos. Cuando comience a modificar código para una iniciativa, debe mover el documento correspondiente a `../en_desarrollo/` y actualizar sus metadatos.

Orden inicial recomendado:

1. consolidar arquitectura, modelo de rutina y contratos;
2. reforzar grabación;
3. construir ejecución determinista;
4. incorporar persistencia y secretos;
5. añadir IA;
6. ampliar automatización con Playwright;
7. endurecer empaquetado y distribución.
